## 🚦 Getting Started


### 3. Ignite the Development Server

```bash
npm run dev
```

## 🎨 Theme Configuration

The project utilizes a centralized theme system located in your CSS variables. To change the "Paint Job," simply update the hex codes in your main CSS file:

```css
:root {
  --color-racing: #ff4d4d;    /* Accent Red */
  --color-mint: #b2ffda;      /* Modern Teal */
  --color-carbon: #1a1a1a;    /* Deep Background */
  --color-platinum: #f5f5f7;  /* Clean Surface */
}

```
